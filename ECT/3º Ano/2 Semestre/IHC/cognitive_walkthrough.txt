Tarefa 1: Um usuário deseja encontrar horários de trem para uma rota específica.

O usuário tentará alcançar a tarefa correta?
Sim, o usuário provavelmente procurará horários de trem se quiser viajar de um local para outro.

O usuário notará que a ação correta está disponível?
Sim, o usuário deve notar o link "Horários" no menu superior, que indicaria que eles podem encontrar horários de trem neste site.

O usuário associará a ação correta com o efeito que está tentando alcançar?
Sim, o usuário deve associar o link "Horários" com o efeito que está tentando alcançar, que é encontrar horários de trem para uma rota específica.

Se a ação correta for realizada, o usuário verá que o progresso está sendo feito?
Sim, se o usuário clicar no link "Horários", ele deverá ser levado a uma página onde poderá inserir suas localizações de partida e chegada para ver horários de trem para sua rota desejada, indicando que o progresso está sendo feito em direção ao alcance de seu objetivo.

Se a ação correta for realizada, o usuário saberá que está se aproximando de seu objetivo?
Sim, se o usuário inserir suas localizações de partida e chegada e ver horários de trem para sua rota desejada, ele deverá saber que está se aproximando de seu objetivo.

Se uma ação incorreta for realizada, o usuário poderá se recuperar do erro?
Sim, se o usuário clicar em um link que não o levar às informações necessárias, ele poderá usar o botão voltar ou o menu de navegação para tentar um caminho diferente para encontrar as informações de que precisa.
